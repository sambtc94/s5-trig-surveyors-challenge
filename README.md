# s5-trig-surveyors-challenge

A browser-based Stage 5 trigonometry game called **Surveyor's Challenge**.

## Run

Open `/home/runner/work/s5-trig-surveyors-challenge/s5-trig-surveyors-challenge/index.html` in a browser.

## Structure

- `index.html` - page markup
- `css/styles.css` - app styles
- `js/main.js` - app entrypoint
- `js/GameApp.js` - game flow and UI controller class
- `js/QuestionService.js` - question generation/solution class
- `js/SceneRenderer.js` - canvas rendering and draw-and-label interaction class
